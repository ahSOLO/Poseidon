Frequently Asked Questions

What is Poseidon?

Poseidon is a free document automation service that allows users to create document templates that can be shared with others through an easy to use form.

How does Poseidon work?

Poseidon generates documents by combining a user-created .DOCX template with information from an online form. For more information about this process, see our getting started page.
